\section{Цель работы}
Исследовать численное решение задачи Коши для ОДУ первого порядка методами Эйлера, Хойна и Рунге--Кутты 4-го порядка, сравнить результаты с аналитическим решением и оценить погрешности.

\section{Постановка задачи}
Для варианта 6 решается задача:
\[
y' = \frac{1}{x^2 + x + 1} - \frac{y}{x + 0.5}, \qquad y(0)=1,\qquad x\in[x_0,b].
\]
Выбраны параметры:
\[
x_0 = 0,\quad b = 2,\quad n = 200,\quad h = \frac{b-x_0}{n} = 0.01.
\]

\section{Теоретическая часть}
\subsection{Задача Коши и сетка}
На отрезке интегрирования вводится равномерная сетка:
\[
x_k = x_0 + kh,\quad k=0,\dots,n,\quad h=\frac{b-x_0}{n}.
\]

\subsection{Метод Эйлера}
\[
y_{k+1} = y_k + h f(x_k, y_k).
\]

\subsection{Метод Хойна}
\[
\hat y_{k+1} = y_k + h f(x_k, y_k),
\]
\[
y_{k+1} = y_k + \frac{h}{2}
\left(
f(x_k, y_k) + f(x_{k+1}, \hat y_{k+1})
\right).
\]

\subsection{Метод Рунге--Кутты 4-го порядка}
\[
k_1 = f(x_k, y_k),\quad
k_2 = f\left(x_k + \frac{h}{2}, y_k + \frac{h k_1}{2}\right),
\]
\[
k_3 = f\left(x_k + \frac{h}{2}, y_k + \frac{h k_2}{2}\right),\quad
k_4 = f(x_k + h, y_k + h k_3),
\]
\[
y_{k+1} = y_k + \frac{h}{6}(k_1 + 2k_2 + 2k_3 + k_4).
\]

\subsection{Оценка ошибок}
\[
\Delta_l = \max_{k=1,\ldots,n}\left|\varphi(x_k)-\tilde\varphi_l(x_k)\right|,
\]
\[
\delta_l = \max_{k=1,\ldots,n}
\frac{\left|\varphi(x_k)-\tilde\varphi_l(x_k)\right|}
{\left|\tilde\varphi_l(x_k)\right|}.
\]

\section{Аналитическое решение}
Перепишем уравнение в линейном виде:
\[
y' + \frac{1}{x+0.5}y = \frac{1}{x^2+x+1}.
\]
Интегрирующий множитель:
\[
\mu(x)=\exp\left(\int\frac{dx}{x+0.5}\right)=x+0.5.
\]
Тогда
\[
\frac{d}{dx}\left[(x+0.5)y\right]
=\frac{x+0.5}{x^2+x+1}
=\frac{1}{2}\frac{2x+1}{x^2+x+1}.
\]
Интегрируя, получаем
\[
(x+0.5)y = \frac{1}{2}\ln(x^2+x+1)+C.
\]
Из условия $y(0)=1$:
\[
0.5 = \frac{1}{2}\ln 1 + C \Rightarrow C=0.5.
\]
Итог:
\[
\varphi(x)=\frac{0.5\ln(x^2+x+1)+0.5}{x+0.5}.
\]
Эта же функция используется в коде как \texttt{ExactSolution}.

\section{Анализ результатов}
Метод Эйлера показывает наибольшую погрешность, так как имеет первый порядок аппроксимации. Метод Хойна использует коррекцию по наклону на правом конце шага и даёт существенно более точный результат (второй порядок). RK4 учитывает четыре промежуточные оценки производной и обеспечивает погрешность порядка $O(h^4)$ по шагу, что подтверждается численными значениями. Уменьшение шага $h$ приводит к снижению ошибок для всех методов, однако скорость убывания различна и соответствует теории.

\text{RK4} \succ \text{Хойн} \succ \text{Эйлер}.
