\section{Цель работы}
Реализовать численное решение задачи Коши для уравнения второго порядка методом Рунге--Кутты 4-го порядка после приведения к системе первого порядка, сравнить результат с аналитическим решением и оценить ошибки.

\section{Постановка задачи}
Для варианта 6 требуется решить:
\[
F(x,y,y',y'')=0,\quad y(x_0)=y_0,\quad y'(x_0)=y_0',\quad x\in[x_0,b],
\]
где
\[
3yy''-3(y')^2=4y(y')^5,\quad y(0)=1,\quad y'(0)=-1.
\]
Выбранные параметры:
\[
x_0=0,\quad b=0.7,\quad n=200,\quad h=\frac{b-x_0}{n}=0.0035.
\]

\section{Теоретическая часть}
\subsection{Задача Коши для уравнения высшего порядка}
Уравнение второго порядка переводится к системе первого порядка, после чего интегрируется векторным одношаговым методом.

\subsection{Переход к системе первого порядка}
\[
u_1=y,\qquad u_2=y',
\]
\[
u_1'=u_2,\qquad u_2'=y''.
\]

\subsection{Векторный RK4}
Для вектора состояния $U_k=(u_{1,k},u_{2,k})^T$:
\[
U_{k+1}=U_k+\frac{h}{6}(K_1+2K_2+2K_3+K_4),
\]
\[
K_1=G(x_k,U_k),
\]
\[
K_2=G\left(x_k+\frac{h}{2},U_k+\frac{hK_1}{2}\right),
\]
\[
K_3=G\left(x_k+\frac{h}{2},U_k+\frac{hK_2}{2}\right),
\]
\[
K_4=G(x_k+h,U_k+hK_3).
\]

\subsection{Формулы погрешностей}
\[
\Delta=\max_{k=1,\ldots,n}\left|\varphi(x_k)-\widetilde{\varphi}(x_k)\right|,
\]
\[
\delta=\max_{k=1,\ldots,n}
\frac{\left|\varphi(x_k)-\widetilde{\varphi}(x_k)\right|}
{\left|\widetilde{\varphi}(x_k)\right|}.
\]

\section{Преобразование уравнения к системе первого порядка}
Из исходного уравнения:
\[
3yy''=3(y')^2+4y(y')^5,
\]
\[
y''=\frac{(y')^2}{y}+\frac{4}{3}(y')^5.
\]
Следовательно, система имеет вид:
\[
u_1'=u_2,\qquad
u_2'=\frac{u_2^2}{u_1}+\frac{4}{3}u_2^5.
\]

\section{Аналитическое решение}
Используется аналитическое решение:
\[
\varphi(x)=\left(1-\frac{4x}{3}\right)^{3/4}.
\]
Проверка начальных условий:
\[
\varphi(0)=1,\qquad
\varphi'(x)=-\left(1-\frac{4x}{3}\right)^{-1/4}\Rightarrow \varphi'(0)=-1.
\]
Так как для действительных значений требуется $1-\frac{4x}{3}>0$, выбираем $b=0.7<0.75$.

\section{Программная реализация}
Программная реализация выполнена на C\# (численное решение и ошибки) и Python (графики и таблицы отчёта). Полный исходный код:

Основные файлы ЛР2:
\begin{itemize}
\item \texttt{lab2/src/Lab2.Core/Variants/Lab2Variant6.cs} --- задание варианта и аналитической функции;
\item \texttt{lab2/src/Lab2.Core/Methods/RungeKutta4SecondOrderMethod.cs} --- RK4 для системы;
\item \texttt{lab2/src/Lab2.Core/Solvers/SecondOrderSolver.cs} --- проход по сетке и ошибки;
\item \texttt{reports/lab2/build\_assets.py} --- графики и таблицы для отчёта.
\end{itemize}

Ключевая схема RK4 для системы первого порядка:
\begin{lstlisting}[language=Python, caption={Общая схема RK4 для системы первого порядка}]
U[0] = [y0, dy0]

for k in range(n):
    K1 = G(x[k], U[k])
    K2 = G(x[k] + h / 2, U[k] + h * K1 / 2)
    K3 = G(x[k] + h / 2, U[k] + h * K2 / 2)
    K4 = G(x[k] + h, U[k] + h * K3)

    U[k + 1] = U[k] + h / 6 * (K1 + 2*K2 + 2*K3 + K4)
\end{lstlisting}
