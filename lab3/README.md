# lab3

здесь лабораторная по pinn для затухающего гармонического осциллятора, вариант 15.

уравнение:

`x'' + 2δx' + ω0² x = 0,  x(0) = 0.9,  x'(0) = -0.2`

параметры варианта 15: `ω0 = 1.2`, `δ = 0.45`, `T = 12`, оцениваемый параметр — `ω0`. режим — слабое затухание (δ < ω0).

что внутри:

- `src` — python/pytorch реализация pinn (прямая и обратная задача)
- `data` — csv с решениями, историей обучения и метриками
- `figures` — графики
- `notebooks` — ноутбук для анализа
- `tests` — pytest на аналитическое решение

как запустить:

```
pip install -r requirements.txt
python -m src.main
```

тесты:

```
pytest -q
```

что считаем:

1. аналитика для underdamped/critical/overdamped (в `src/analytical.py`)
2. прямая задача — pinn аппроксимирует решение на `[0, T]`, сверяем с аналитикой
3. на точки накладываем гауссовский шум `N(0, 0.05²)`
4. обратная задача — `ω0` делаем обучаемым параметром, `δ` и нач. условия фиксированы, обучаем pinn на зашумленных данных

результаты на варианте 15:

- прямая: max_abs_error ≈ 4e-5, rmse ≈ 1.3e-5
- обратная: ω0_оценка ≈ 1.215 при истинном 1.2, отн. ошибка ~1.2%
