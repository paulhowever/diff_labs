# lab3

здесь лабораторная по pinn для затухающего гармонического осциллятора, вариант 6.

уравнение:

`x'' + 2δx' + ω0² x = 0,  x(0)=1,  x'(0)=0`

параметры варианта 6: `ω0 = 2.0`, `δ = 0.2`, `T = 15`, оцениваемый параметр — `δ`. режим — слабое затухание (δ < ω0).

что внутри:

- `src` — python/pytorch реализация pinn (прямая и обратная задача)
- `data` — csv с решениями, историей обучения и метриками
- `figures` — графики
- `notebooks` — ноутбук для анализа
- `tests` — pytest на аналитическое решение

как запустить:

```
pip install -r requirements.txt
python -m src.main
```

тесты:

```
pytest -q
```

что считаем:

1. аналитика для underdamped/critical/overdamped (в `src/analytical.py`)
2. прямая задача — pinn аппроксимирует решение на `[0, T]`, сверяем с аналитикой
3. на точки накладываем гауссовский шум `N(0, 0.05²)`
4. обратная задача — `δ` делаем обучаемым параметром, ω0 и нач. условия фиксированы, обучаем pinn на зашумленных данных

результаты на варианте 6:

- прямая: max_abs_error ≈ 1e-4, rmse ≈ 5e-5
- обратная: δ_оценка ≈ 0.198 при истинном 0.2, отн. ошибка ~1%
