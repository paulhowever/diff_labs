# Lab 1 (Вариант 6)

Численное решение задачи Коши 1-го порядка:

- `y' = 1/(x^2 + x + 1) - y/(x + 0.5)`
- `y(0) = 1`

## Структура

- `src` — C# решение (методы Эйлера, Хойна, RK4)
- `tests` — численные тесты
- `data` — результаты вычислений в CSV
- `notebooks` — Python-анализ и графики

## Быстрый старт

1. Сгенерировать данные:
   - `dotnet run --project src/Lab1.App`
2. Запустить тесты:
   - `dotnet test tests/Lab1.Tests`
3. Открыть ноутбук:
   - `notebooks/analysis.ipynb`

Единый формат CSV описан в `../DATA_CONTRACT.md`.

## Запуск из корня репозитория

- `./run_all.sh` — посчитать обе лабы и прогнать тесты
- `./run_all.sh --open-notebooks` — то же + открыть ноутбуки
- `make all` — shortcut для полного прогона
