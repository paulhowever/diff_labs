# lab1

здесь лабораторная по задаче коши первого порядка, вариант 6.

уравнение такое:

`y' = 1/(x^2 + x + 1) - y/(x + 0.5), y(0) = 1`

что внутри:

- `src` — реализация методов эйлера, хойна и rk4 на c#
- `tests` — численные тесты
- `data` — csv с результатами
- `notebooks` — python ноутбук для графиков и таблиц

параметры расчета в текущей версии:

- `x0 = 0`
- `b = 2`
- `n = 200`
- `h = (b - x0)/n`

формат выходных csv общий с lab2 и описан в `../DATA_CONTRACT.md`.
