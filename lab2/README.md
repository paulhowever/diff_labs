# lab2

здесь лабораторная по задаче коши для уравнения второго порядка, вариант 6.

исходное уравнение:

`3yy'' - 3(y')^2 = 4y(y')^5, y(0)=1, y'(0)=-1`

что внутри:

- `src` — c# реализация приведения к системе первого порядка и rk4 для системы
- `tests` — численные тесты
- `data` — csv с результатами
- `notebooks` — python ноутбук для графиков и таблиц

параметры расчета в текущей версии:

- `x0 = 0`
- `b = 0.7`
- `n = 200`
- `h = (b - x0)/n`

формат выходных csv общий с lab1 и описан в `../DATA_CONTRACT.md`.
